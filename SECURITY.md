# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a vulnerability

If you discover a security issue, please **do not** open a public GitHub issue.

Instead, report it privately to the repository maintainers. Include:

- A description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix, if you have one

Recovery reads raw disk devices and writes recovered files to a user-chosen destination. Reports related to path traversal, unsafe file writes, privilege escalation, or remote code execution in the local web UI are especially welcome.

We aim to acknowledge reports within a few business days.
